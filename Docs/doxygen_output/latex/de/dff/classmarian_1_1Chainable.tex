\subsection{Detailed Description}
\subsubsection*{template$<$class Data\+Type$>$\\*
struct marian\+::\+Chainable$<$ Data\+Type $>$}

Abstraction of an element in a computation graph for which a derivative can be calculated. 

The name of this class comes from the fact that this element is \href{https://en.wikipedia.org/wiki/Function_composition}{\tt composable} (aka chainable) in the context of the \href{https://en.wikipedia.org/wiki/Chain_rule}{\tt chain rule of calculus}.

Given that context, in the documentation for this class, the following notation is used\+:
\begin{DoxyItemize}
\item Given an expression graph of composed functions, $y$ refers to the final value resulting from evaluating the entire graph
\item $w_i$ refers to the partial result of evaluating the expression subgraph rooted at the {\itshape i}-\/th \hyperlink{classmarian_1_1Chainable}{Chainable} element
\item $\bar{w}_i$ refers to the \href{https://en.wikipedia.org/wiki/Automatic_differentiation#Reverse_accumulation}{\tt adjoint} of $w_i$, where $\bar{w}_i$ is defined as the partial derivative of $y$ with respect to $w_i$, or formally $\bar{w}_i = \frac{\partial y}{\partial w_i}$ 
\end{DoxyItemize}
