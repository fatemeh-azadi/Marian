\subsection{Detailed Description}
Represents a \href{https://en.wikipedia.org/wiki/Rectifier_(neural_networks)}{\tt rectified linear} node in an expression graph.

This node implements the \href{https://en.wikipedia.org/wiki/Activation_function}{\tt activation function} $f(x) = \max(0, x)$ and its derivative\+:

\[ f^\prime(x) = \begin{cases} 0 & \text{if } x \leq 0 \\ 1 & \text{if } x > 0 \end{cases} \]
