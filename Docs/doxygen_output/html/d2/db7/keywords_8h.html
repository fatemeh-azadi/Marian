Defines a preprocessor macro that can be used to construct an appropriately templated instance of the Keyword class.

Parameters
nameThis preprocessor argument specifies the variable name of constructed keyword instance
value_typeThis preprocessor argument specifies the type of the value to be stored in the keyword instance

For example, the invocation KEY(axis, int) is equivalent to the following: 

Keyword<23423, int> axis;

Where 23423 is the hypothetical hash code returned at compile time by COMPILE_TIME_CRC32_STR("axis")
