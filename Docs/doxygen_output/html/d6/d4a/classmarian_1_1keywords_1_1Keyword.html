Detailed Description

template<unsigned key, typename Value>
class marian::keywords::Keyword< key, Value >

Represents a named keyword capable of storing a single value.

This class is used to emulate keyword arguments to functions, such as those found in Python.

It is expected that users of this class will not explicitly create instances of this class.

Rather, it is expected that users will use the KEY(name, value_type) macro.

For example, the invocation KEY(batch_size, int) will construct a new instance of this class called batch_size whose value_type is int.

Now assume a function called foo() that requires a single Keyword as its parameter.

foo(batch_size=200)

This code fragment causes an invocation of this class's operator= method, which returns a new instance of the Keyword class. This new instance will have a value of 200.
