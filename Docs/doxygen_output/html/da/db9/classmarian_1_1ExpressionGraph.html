void marian::ExpressionGraph::backward ( )
inline

Perform the backward pass of algorithmic differentiation (AD) on this graph.

This pass traverses the nodes of this graph in reverse of the order they were created; as each node is traversed, its set_zero_adjoint() method is called.

Once this has been performed for all nodes, this pass again traverses the nodes, again in reverse creation order; as each node is traversed, its backward() method is called.

After this method has successfully completed, and that all backward pass computations have been performed.

void marian::ExpressionGraph::forward ( )
inline

Perform the forward pass of algorithmic differentiation (AD) on this graph.

This pass traverses the nodes of this graph in the order they were created; as each node is traversed, its allocate() method is called.

Once allocation is complete for all nodes, this pass again traverses the nodes, in creation order; as each node is traversed, its forward() method is called.

After this method has successfully completed, it is guaranteed that all node allocation has been completed, and that all forward pass computations have been performed.
